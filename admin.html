Configura Firebase

Il backoffice e la vetrina usano Firebase per database e login. Una volta sola:

  1. Vai su console.firebase.google.com → Crea progetto (es. amidenim).
  2. Menu Build → Realtime Database → Crea database → zona europe-west1 → modalità bloccata.
  3. Menu Build → Authentication → Inizia → abilita Email/Password. Poi tab Users → Aggiungi utente: crea il tuo account admin (email + password).
  4. Impostazioni progetto (ingranaggio) → in fondo Le tue app → icona web </> → registra l'app → copia l'oggetto firebaseConfig.
  5. Incolla quei valori dentro FIREBASE_CONFIG in cima a admin.html e index.html (identici).
  6. In Realtime Database → tab Regole, incolla le regole qui sotto e pubblica.
